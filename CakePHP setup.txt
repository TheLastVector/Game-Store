How to create a CakePHP Project :
1- Open the Command-line
2- Position yourself in the repository you want your project to be created
3- Execute this command (replace "my_app_name" with the name of your project) : 
	composer self-update && composer create-project --prefer-dist cakephp/app my_app_name

Cooking recepies :
cd /path/to/our/app/bin

cake bake model tableName
cake bake controller tableName
cake bake template tableName

or,

cake bake all tableName
